CodeKashi - Eternal Code Temples

This is the foundational architecture for the CodeKashi portfolio, built with Vite, React, Tailwind CSS, Framer Motion, and React Three Fiber (for Three.js).

How to Run This Project

Save All Files: Make sure you have all the files from this project (package.json, vite.config.js, tailwind.config.js, etc.) in a single folder.

Open Your Terminal: Navigate into that folder.

Install Dependencies: Run the following command. This will read your package.json and install React, Three.js, Tailwind, and all other required libraries.

npm install


Run the Development Server: Once installation is complete, run this command:

npm run dev


View Your Site: Your terminal will give you a local URL (usually http://localhost:5173/). Open that in your browser.

You will have the full, working website, built correctly and ready for you to add your other 3D models and features.
